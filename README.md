### Glassmorphism Cards
All cards use the glassmorphism effect with:

- GSAP-powered animations

## ğŸ¨ Customization

### Colors
The primary color scheme can be modified in `styles.css`:
```css
:root {
  --primary-blue: #0d6efd;
  --primary-purple: #6f42c1;
  --dark-bg: #000000;
```
